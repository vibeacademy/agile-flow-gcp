## Plan UGC Campaign

Launch the growth-marketing-strategist agent to design a user-generated content campaign.

## Instructions

Guide the user through UGC campaign planning by asking these questions ONE AT A TIME:

### Question 1
```
What type of UGC campaign do you want to run?

1. Social media challenge/hashtag campaign
2. Review/testimonial collection
3. Photo/video contest
4. User spotlight/case study program
5. Community showcase (share your work)
6. Ambassador content program
7. Not sure - recommend based on my product

Enter a number (1-7):
```

### Question 2
```
What content format works best for your product?

1. Photos
2. Short-form video (TikTok, Reels)
3. Long-form video (YouTube)
4. Written reviews/testimonials
5. Screenshots/screen recordings
6. Multiple formats

Enter a number (1-6):
```

### Question 3
```
What incentives can you offer participants?

1. Feature their content on our channels
2. Product discounts or free upgrades
3. Cash prizes or gift cards
4. Physical prizes (products, swag)
5. Recognition only (no material incentive)
6. Combination (please describe)

Enter a number (1-6) or describe:
```

### Question 4
```
How long should the campaign run?

1. One-time event (1-2 weeks)
2. Monthly recurring
3. Ongoing/evergreen program
4. Tied to specific event/launch

Enter a number (1-4):
```

### Question 5
```
Where will you promote and collect submissions?

1. Instagram only
2. TikTok only
3. Multiple social platforms
4. Email to existing users
5. In-app prompt
6. All channels

Enter a number (1-6):
```
